Send Daily Summary

💾
Backup Database
Dumps MySQL to a timestamped .sql file in the backups folder. Keeps the 5 most recent backups.
🗄️ Database Migrations (Flask-Migrate)
Run these in a PythonAnywhere Bash console when you change models.py:
# First time only:
flask --app app db init
flask --app app db migrate -m "initial"
flask --app app db upgrade
# After every models.py change:
flask --app app db migrate -m "describe change"
flask --app app db upgrade
